== Description == 

Customer Conversion Tracker provides useful data on your customers' checkout experience in order to increase conversions and reduce abandoned carts. The plugin provides real-time data on how your stores customers are flowing through the process of purchasing an item.

- 'Added to Cart' shows the number of site visitors that have added at least one item to their cart.
- 'Reached Checkout' shows the number of customers who have added an item to their cart and reached your stores checkout stage.
- 'Purchased' shows the number of those customers who successfully completed their purchase.

You can then test solutions to improve your customer checkout experience in real-time and view the effects over a custom time period.

This is an essential tracking tool for anyone running a Woocommerce store.

== Installation ==

1. Ensure you have the Woocommerce for Wordpress plugin and have activated it in your plugins page. 
2. Install Customer Conversion Tracker plugin and activate it via the plugin page. 

== Frequently Asked Questions ==

No, Customer Conversion Tracker requires Woocommerce to be installed before it is installed.
